# Admin Dashboard

A fully responsive admin dashboard built with React, Vite, Tailwind CSS, and Recharts.

## Features
- Dashboard overview with key metrics (Revenue, Orders, Customers, Growth)
- Revenue & Expenses area chart with period filters
- Traffic sources donut chart
- Recent orders table with status badges
- Top products list with performance indicators
- Responsive sidebar navigation

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
